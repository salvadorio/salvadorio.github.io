Yoshi Sodeoka inspired Projection Mapping

     w/ Prof. Lauren Petty

     For this project, I worked with Sam to make something inspired by the work of Yoshi Sodeoka. She took pictures of DUMBO and edited them, and I wrote python code to calibrate a random webcam to watch the output of a projector. When someone passes in front of the projection, the webcam would detect motion and a location using OpenCV and distort that area of the projected image. This would have 2 different trippy feedback-loop effects. There was some complicated pixel bitwise operation stuff going on layered with mask erosion, but also, any lasting impact you would have on the projection would create more movement after you're done, and more movment detected would cause more distortion, to an extent we calibrated. I generally really like image processing stuff, so it was a fun project.
